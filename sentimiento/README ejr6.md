
# Análisis de sentimiento con Node.js

Script de Node.js que analiza el sentimiento de varios textos 
usando el módulo de terceros "sentiment".

## Funcionalidades

- Analiza si un texto es positivo, negativo o neutro
- Muestra la puntuación numérica de cada texto
- Cuanto mayor es la puntuación, más positivo es el texto

## Tecnologías usadas

- Node.js
- npm
- Módulo sentiment (npm install sentiment)

## Archivos

- index.js — código principal del análisis
- package.json — configuración del proyecto y dependencias

## Cómo ejecutarlo

npm install
npm start
